# Osome-Chat-Logs

Hier kommen die Verläufe aus der **Osome-App** rein — als **Text** (kopieren/einfügen),
**nicht** als Link. Hinter dem Osome-Login kann Claude nicht selbst nachsehen.

## So geht's
1. In der Osome-App (my.osome.com) das jeweilige Ticket öffnen.
2. Den Chat-Verlauf markieren und kopieren.
3. Pro Vorgang eine Datei anlegen, z. B.:
   - `bank-account-opening.md`
   - `bookkeeping-details.md`
   - `post-incorporation-documents.md`
   - `incorporate-company.md`
4. Datum oben notieren, Text einfügen, committen.

Danach liest Claude bei jeder Session den vollständigen Stand mit.
