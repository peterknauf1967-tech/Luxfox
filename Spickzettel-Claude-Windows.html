🦊 Spickzettel — Claude & Windows Befehle

Die wichtigsten Befehle für die tägliche Arbeit mit Claude Code unter Windows

⌨️ Claude-Befehle (im Chat eintippen)

/helpListe aller Befehle anzeigen
/clearUnterhaltung neu starten (Verlauf leeren)
/configEinstellungen öffnen (Modell, Design)
/fastSchnell-Modus ein-/ausschalten
/initProjekt-Infos für Claude anlegen
/reviewCode/Änderungen prüfen lassen
/costVerbrauch der Sitzung anzeigen
/loginAn- / Abmelden
EscAktuelle Aktion abbrechen / stoppen
↑ / ↓Frühere Eingaben durchblättern
Shift + EnterNeue Zeile (ohne abzusenden)

💬 So formulierst du gute Aufträge

„Erkläre mir…"Lass dir alles in einfachen Worten erklären
„Baue/Füge … hinzu"Konkret sagen, was du willst
„Mach … rückgängig"Letzte Änderung zurücknehmen
„Zeig mir das Ergebnis"Vorschau / Test der Änderung
„Speichere & lade hoch"Änderungen committen & pushen

🪟 Windows-Befehle (Eingabeaufforderung)

cd OrdnerIn einen Ordner wechseln
cd ..Eine Ebene nach oben
dirInhalt des Ordners anzeigen
clsBildschirm leeren
mkdir NameNeuen Ordner erstellen
copy a bDatei kopieren
move a bDatei verschieben / umbenennen
del DateiDatei löschen
type DateiInhalt einer Datei anzeigen
start .Aktuellen Ordner im Explorer öffnen
where dateiPfad zu einem Programm finden
ipconfigNetzwerk-/IP-Infos anzeigen

🔧 Git-Befehle (Code verwalten)

git statusWas wurde geändert?
git add .Alle Änderungen vormerken
git commit -m "Text"Änderungen speichern (mit Notiz)
git pushHochladen zu GitHub
git pullNeueste Version holen
git logVerlauf der Änderungen

🇩🇪 Sonderzeichen auf der QWERTZ-Tastatur (mit AltGr)

Die AltGr-Taste liegt rechts neben der Leertaste. Halte sie gedrückt und tippe die zweite Taste.

@ (At / E-Mail)AltGr + Q
\ (Backslash / Pfade)AltGr + ß
| (Pipe / senkr. Strich)AltGr + <
~ (Tilde)AltGr + +
€ (Euro)AltGr + E
{ (geschw. Klammer auf)AltGr + 7
} (geschw. Klammer zu)AltGr + 0
[ (eckige Klammer auf)AltGr + 8
] (eckige Klammer zu)AltGr + 9
/ (Schrägstrich)Umschalt + 7
💡 Tipp: Du musst dir nichts merken — frag Claude einfach im Klartext, z. B. „Wie öffne ich meinen Projekt-Ordner?" oder „Lade meine Änderungen hoch". Claude führt die passenden Befehle für dich aus. Dieser Zettel hilft dir nur, mitzulesen, was passiert.
